# Generated Feedback Reports

This folder contains student feedback reports and class summary reports generated by `tools/generate_feedback.py`.

## Folder Structure

```
generated/
├── student-reports/     # Individual student feedback (one file per student per assessment)
└── class-reports/       # Class-wide summary reports (one file per assessment)
```

## File Naming

Student reports: `feedback-[student_id]-[assessment_id].md`
Class reports: `class-summary-[assessment_id].md`

## How to Generate

Run from the repository root:

```bash
python tools/generate_feedback.py
```

Output files will appear in `feedback/generated/`.
