# Multiple Choice Questions (MCQ)

Place AP-style MCQ sets here.

## Naming Convention
`mcq-[unit or topic]-[date or version].[ext]`

**Examples:**
- `mcq-unit03-chain-rule-2026-10.pdf`
- `mcq-ap-review-full-2027-04.pdf`

## Notes
- Label each question with a topic and skill
- Include an answer key for each set
- Track results per question in `data/results.csv`
