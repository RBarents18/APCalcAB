# Free Response Questions (FRQ)

Place AP-style FRQ sets here.

## Naming Convention
`frq-[unit or topic]-[date or version].[ext]`

**Examples:**
- `frq-unit07-differential-equations-2026-11.pdf`
- `frq-ap-review-set01-2027-03.pdf`

## Notes
- Use AP scoring guidelines when creating rubrics
- Reference standard AP FRQ format: 3–4 parts, written justification required
