# Rubrics

Place scoring rubrics here for quizzes, tests, and FRQs.

## Naming Convention
`rubric-[assessment-id]-[short-title].[ext]`

**Examples:**
- `rubric-U1Q1-limits-quiz.pdf`
- `rubric-U7FRQ1-slope-fields.pdf`

## Notes
- Rubrics should map clearly to skills in `data/standards_map.csv`
- AP FRQ rubrics should follow College Board scoring guide format
