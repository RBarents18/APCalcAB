# Student Review Plans

This folder contains individual student AP Exam review plans generated by `tools/build_review_plan.py`.

## File Naming

`review-plan-[student_id]-[date].md`

**Example:** `review-plan-1001-2027-04-22.md`

## Contents of Each Plan

Each student review plan includes:
- Current overall performance summary
- Unit-by-unit skill summary
- Top 3–5 priority review areas
- Recommended practice resources by skill
- AP Exam tips relevant to the student's weak areas
