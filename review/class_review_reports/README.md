# Class Review Reports

This folder contains whole-class AP review priority reports generated by `tools/build_review_plan.py`.

## File Naming

`class-review-priority-[date].md`

**Example:** `class-review-priority-2027-04-22.md`

## Contents of Each Report

Each class review report includes:
- Class-wide performance summary by unit
- High, Medium, and Maintenance priority topic lists
- Suggested review schedule
- FRQ and MCQ focus areas for the class
- Notes on which students need additional individual support
