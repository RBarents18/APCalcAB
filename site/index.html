Curriculum

8 AP Calculus AB Units

Organized around the official AP Calculus AB unit framework.

Unit 1

Limits and Continuity

The conceptual foundation of calculus—limits, continuity, discontinuities, and the Intermediate Value Theorem.

Foundational
Unit 2

Differentiation: Definition & Fundamental Properties

Derivative definition, basic rules (power, product, quotient), and derivatives of trig/exp/log functions.

Core
Unit 3

Differentiation: Composite, Implicit & Inverse Functions

Chain rule, implicit differentiation, inverse trig derivatives, and higher-order derivatives.

Core
Unit 4

Contextual Applications of Differentiation

Rates of change in context, related rates, linearization, L'Hôpital's Rule, and motion problems.

Core
Unit 5

Analytical Applications of Differentiation

Mean Value Theorem, increasing/decreasing intervals, critical points, concavity, and optimization.

Core
Unit 6

Integration and Accumulation of Change

Riemann sums, definite integrals, Fundamental Theorem of Calculus, and u-substitution.

Core
Unit 7

Differential Equations

Slope fields, Euler's method, separation of variables, and exponential growth and decay models.

Important
Unit 8

Applications of Integration

Average value, particle motion, area between curves, and volumes using disks, washers & cross-sections.

Core

AP Exam Prep

Data-Driven Review Workflow

Use class assessment data to drive targeted AP Exam review— spend time where it matters most.

1

Collect & Import Results

Add student scores to data/results.csv after each assessment.

2

Run Analysis

Execute python tools/analyze_results.py for skill-level performance summaries.

3

Generate Feedback

Run tools/generate_feedback.py to create per-student reports in feedback/generated/.

4

Build Review Plan

Run tools/build_review_plan.py to rank units and skills by urgency.

5

Export Site Data

Run python tools/export_site_data.py to update the visual review dashboard.

Priority Categories

Review Priorities

🔴

High Priority

Low class performance on high-AP-relevance skills. These are the first topics addressed in review.

🟡

Medium Priority

Moderate performance or recurring errors. Targeted small-group sessions or quick re-teaching.

🟢

Maintenance

Strong performance — still reviewed through spiral mixed-practice sets so skills stay sharp.

Data Workflow

Assessments & Python Analytics

The repository uses structured CSVs and stdlib Python scripts—no heavy dependencies required.

📄 assessments.csv

Metadata for every quiz, test, FRQ, and MCQ—title, unit, date, and total points.

🗂️ question_bank.csv

Maps each question to a topic, skill, and point value for granular tracking.

📈 results.csv

Per-student, per-question performance data powering all analysis scripts.

🗺️ standards_map.csv

Links topics and skills to AP exam weight categories and review priority levels.
