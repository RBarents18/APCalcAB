# Unit 7: Differential Equations — Skills

## AP Calculus AB Skill Categories

### Skill 1: Implementing Mathematical Processes
- Verify a solution by substituting into the differential equation
- Sketch slope field tick marks by evaluating dy/dx at grid points
- Apply Euler's method: y_(n+1) = y_n + f(x_n, y_n)·Δx
- Separate variables and integrate both sides to find a general solution
- Apply an initial condition to solve for the constant of integration
- Write the general solution for dy/dt = ky as y = Ce^(kt)

### Skill 2: Connecting Representations
- Match a DE to its slope field by analyzing sign and behavior
- Sketch a particular solution curve through a given point on a slope field
- Connect algebraic solution to graphical behavior shown in the slope field

### Skill 3: Justification
- Justify the solution method for a separable DE
- Verify that a particular solution satisfies both the DE and the initial condition
- Explain what the constant k means in an exponential model

### Skill 4: Communication and Notation
- Write differential equations in standard form
- Show all separation of variables steps clearly (move terms before integrating)
- Interpret and communicate exponential model results in context with units

## Key Vocabulary
- Differential equation, general solution, particular solution
- Slope field, solution curve, initial condition
- Euler's method, step size
- Separation of variables
- Exponential growth, exponential decay, growth constant k
- Logistic growth, carrying capacity, inflection point
