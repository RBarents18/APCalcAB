# Unit 5: Analytical Applications of Differentiation — Skills

## AP Calculus AB Skill Categories

### Skill 1: Implementing Mathematical Processes
- Find critical points by setting f′(x) = 0 or identifying where f′(x) is undefined
- Construct a sign chart for f′ and f″ to analyze behavior
- Apply first derivative test: sign change of f′ determines local min/max
- Apply second derivative test: sign of f″ at a critical point determines concavity
- Use the Candidates Test to find global extrema on [a, b]
- Set up an optimization equation, reduce to one variable, and minimize/maximize

### Skill 2: Connecting Representations
- Connect the graph of f to the graphs of f′ and f″
- Translate between graphical behavior and derivative sign information
- Identify inflection points on a graph of f″

### Skill 3: Justification
- Use the Mean Value Theorem with a written justification
- Justify local and global extrema using appropriate tests
- Justify inflection points using sign change of f″
- Write a complete optimization justification (show it is a max/min, not just critical)

### Skill 4: Communication and Notation
- Write justification statements for increasing/decreasing in complete sentences
- Clearly label concavity intervals and inflection points
- Communicate optimization conclusions with units and context

## Key Vocabulary
- Mean Value Theorem, Extreme Value Theorem, Rolle's Theorem
- Critical point, local minimum, local maximum, global minimum, global maximum
- First derivative test, second derivative test, Candidates Test
- Increasing, decreasing, concave up, concave down, inflection point
- Optimization, constraint equation, objective function
