# Unit 5: Analytical Applications of Differentiation — Learning Objectives

By the end of this unit, students will be able to:

1. State and apply the Mean Value Theorem.
2. Apply Rolle's Theorem as a special case of MVT.
3. Apply the Extreme Value Theorem to guarantee the existence of extrema on a closed interval.
4. Define critical points and find them algebraically.
5. Use the first derivative test to classify critical points as local minima, maxima, or neither.
6. Use the second derivative test to classify critical points.
7. Determine intervals where a function is increasing or decreasing.
8. Define concavity and identify intervals where a function is concave up or concave down.
9. Find and justify points of inflection.
10. Find global (absolute) extrema on closed intervals using the Candidates Test.
11. Set up and solve optimization problems with a single variable.
12. Sketch a complete graph of a function using first and second derivative analysis.
