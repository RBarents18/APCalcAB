# Unit 5: Analytical Applications of Differentiation

## Description
Students use derivatives to analyze function behavior, solve optimization problems, and justify conclusions about graphs and intervals.

## Major Topics
- Mean Value Theorem and Rolle's Theorem
- Extreme Value Theorem
- Increasing and decreasing behavior (first derivative test)
- Critical points and local extrema
- Concavity and inflection points (second derivative test)
- Global extrema on closed intervals
- Curve sketching using calculus
- Optimization

## AP Exam Relevance
Analytical applications of derivatives are among the most heavily tested areas. Both analytical and graphical reasoning are required in the FRQ sections.

## Evidence of Mastery
Students can:
- Apply the Mean Value Theorem to guarantee the existence of a value
- Find and classify critical points using the first and second derivative tests
- Determine intervals of increase/decrease and concavity
- Identify local and global extrema
- Set up and solve optimization problems
- Sketch a curve using derivative analysis
