# Unit 6: Integration and Accumulation of Change — Learning Objectives

By the end of this unit, students will be able to:

1. Interpret the definite integral as a limit of Riemann sums.
2. Calculate left, right, midpoint, and trapezoidal Riemann sums to approximate integrals.
3. Use integral notation and define the definite integral on an interval [a, b].
4. Apply properties of definite integrals (linearity, additivity, symmetry).
5. Evaluate definite integrals using the Fundamental Theorem of Calculus, Part 2.
6. Find the derivative of an accumulation function using FTC, Part 1.
7. Interpret an accumulation function and its rate of change in context.
8. Find antiderivatives of basic functions using reverse differentiation rules.
9. Evaluate indefinite integrals using u-substitution.
10. Evaluate definite integrals using u-substitution with updated limits.
11. Interpret the definite integral as net change or total accumulated value.
12. Use the net change theorem to answer applied questions about displacement or accumulation.
