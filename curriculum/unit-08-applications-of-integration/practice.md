# Unit 8: Applications of Integration — Practice

## Starter Practice Problems

### Average Value
1. Find the average value of f(x) = x² on [0, 3].
2. A car's velocity is v(t) = 60 - 4t mph. Find the average velocity from t = 0 to t = 10.
3. The temperature on a day is T(t) = 70 + 10sin(πt/12) from t = 0 to t = 12 hours. Find the average temperature.

### Net Change and Particle Motion
4. A particle's velocity is v(t) = t² - 4t + 3 on [0, 4]. Find:
   - Net displacement
   - Total distance traveled

### Area Between Curves
5. Find the area of the region bounded by f(x) = x² and g(x) = x + 2.
6. Find the area enclosed by y = sin(x) and y = cos(x) on [π/4, 5π/4].
7. Find the area of the region bounded by y = x² - 2 and y = x.

### Volumes: Known Cross Sections
8. The base of a solid is the region bounded by y = √x and y = 0 on [0, 4]. Cross sections perpendicular to the x-axis are squares. Set up and evaluate the volume integral.
9. Same region as above, but cross sections are equilateral triangles. Set up the integral.

### Disk and Washer Method
10. Find the volume of the solid formed when y = √x on [0, 4] is rotated about the x-axis. (Disk method)
11. Find the volume of the solid formed when the region bounded by y = x and y = x² is rotated about the x-axis. (Washer method)
12. Set up the integral for the volume formed when y = x² + 1 on [0, 2] is rotated about y = -1. (Washer method)

## Activity Ideas
- Desmos visualization: graph curves and shade bounded regions before computing area
- Solid of revolution demonstration: use physical models or 3D visualization tools
- AP FRQ area/volume: complete a multi-part FRQ with area and volume sub-parts
- Average value meaning: graph f, draw the horizontal average value line, and compare areas
