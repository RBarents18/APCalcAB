# Unit 8: Applications of Integration — Learning Objectives

By the end of this unit, students will be able to:

1. Calculate the average value of a function on a closed interval.
2. Interpret the average value in context (e.g., average temperature, average velocity).
3. Use the net change theorem to find displacement and total distance from a velocity function.
4. Set up an integral for the area between two curves and identify correct limits of integration.
5. Find the area of a region bounded by two curves by setting up a difference of integrals.
6. Determine when to integrate with respect to x versus with respect to y.
7. Set up integrals for volumes of solids with square or rectangular cross sections.
8. Apply the disk method to find the volume of a solid of revolution.
9. Apply the washer method when the region does not touch the axis of rotation.
10. Correctly identify the outer and inner radii for the washer method.
11. Set up and evaluate complete area and volume problems from a description or graph.
12. Justify the set-up of an integral for volume or area using correct geometric reasoning.
