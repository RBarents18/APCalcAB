# Unit 8: Applications of Integration

## Description
Students apply definite integrals to solve problems involving average value, accumulated change, area between curves, and volume of solids.

## Major Topics
- Average value of a function
- Connecting position, velocity, and acceleration using integrals
- Net change theorem and particle motion
- Area between two curves
- Volumes of solids with known cross sections
- Volumes of solids of revolution using the disk and washer methods

## AP Exam Relevance
Applications of integration are among the most frequently tested areas on the AP exam. Area and volume problems appear on almost every FRQ section.

## Evidence of Mastery
Students can:
- Calculate the average value of a function on an interval
- Find net displacement and total distance using integrals
- Set up and evaluate integrals for the area between two curves
- Set up integrals for volumes using cross sections (squares, rectangles)
- Set up and evaluate disk and washer method integrals for volumes of revolution
- Correctly identify limits of integration for area and volume problems
