# Unit 3: Differentiation: Composite, Implicit, and Inverse Functions — Skills

## AP Calculus AB Skill Categories

### Skill 1: Implementing Mathematical Processes
- Identify and apply the chain rule: d/dx[f(g(x))] = f′(g(x)) · g′(x)
- Differentiate composite functions involving trig, exponential, and logarithmic forms
- Perform implicit differentiation, treating y as a function of x
- Apply the inverse function derivative formula: (f⁻¹)′(x) = 1 / f′(f⁻¹(x))
- Differentiate arcsin(x), arctan(x), and related forms

### Skill 2: Connecting Representations
- Connect the chain rule to the idea of rates of change in a chain of dependent variables
- Relate implicit differentiation to the slope of a tangent line on a curve
- Use a graph of f to reason about the derivative of f⁻¹

### Skill 3: Justification
- Justify the use of chain rule in a multi-step derivative
- Show the steps of implicit differentiation clearly and in order
- Verify a derivative formula using the definition or another rule

### Skill 4: Communication and Notation
- Write d/dx notation correctly in implicit differentiation steps
- Label outer and inner functions when applying chain rule
- Express higher-order derivatives using appropriate notation (f″, d²y/dx²)

## Key Vocabulary
- Chain rule, composite function, inner function, outer function
- Implicit differentiation, explicitly defined, implicitly defined
- Inverse function, inverse trigonometric function
- Higher-order derivative, second derivative
