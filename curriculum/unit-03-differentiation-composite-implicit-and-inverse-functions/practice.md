# Unit 3: Differentiation: Composite, Implicit, and Inverse Functions — Practice

## Starter Practice Problems

### Chain Rule
1. Differentiate: f(x) = (3x² + 1)⁵
2. Differentiate: g(x) = sin(4x³)
3. Differentiate: h(x) = e^(x² - 2x)
4. Differentiate: y = ln(cos x)
5. Differentiate: f(x) = (x² + 1)³ · e^(2x)  *(combines chain and product rules)*

### Implicit Differentiation
6. Find dy/dx for: x² + y² = 25
7. Find dy/dx for: x³ + y³ = 6xy
8. Find the slope of the tangent line to x² + xy + y² = 7 at the point (1, 2).
9. Find d²y/dx² for x² + y² = 16.

### Inverse Functions
10. If f(3) = 7 and f′(3) = 4, find (f⁻¹)′(7).
11. Differentiate: f(x) = arctan(2x)
12. Differentiate: g(x) = arcsin(x²)
13. Differentiate: h(x) = x · arctan(x)

### Higher-Order Derivatives
14. Find f″(x) for f(x) = 3x⁴ - 2x² + 5x.
15. If s(t) = t⁴ - 6t², find velocity s′(t) and acceleration s″(t).

## Activity Ideas
- Chain rule visual: diagram decomposing composite functions into layers
- Implicit differentiation stations: different curves with tangent line tasks
- Inverse function exploration: verify (f⁻¹)′ graphically and algebraically
- AP FRQ mini: multi-rule derivative problems in context
