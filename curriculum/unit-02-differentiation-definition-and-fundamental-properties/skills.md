# Unit 2: Differentiation: Definition and Fundamental Properties — Skills

## AP Calculus AB Skill Categories

### Skill 1: Implementing Mathematical Processes
- Set up and evaluate the limit definition of the derivative
- Apply the power rule, product rule, and quotient rule
- Differentiate sin(x), cos(x), eˣ, and ln(x)
- Determine non-differentiable points (corners, cusps, vertical tangents, discontinuities)

### Skill 2: Connecting Representations
- Match a function graph to its derivative graph
- Estimate derivative values from a table of function values
- Connect slope of tangent line to the derivative value at a point
- Relate average rate of change (secant slope) to instantaneous rate of change (derivative)

### Skill 3: Justification
- Justify differentiability using the limit definition
- Explain why differentiability implies continuity but not the converse
- Justify tangent line equations using the point-slope form

### Skill 4: Communication and Notation
- Use prime notation (f′) and Leibniz notation (dy/dx) correctly
- Express derivatives in context with appropriate units
- Clearly communicate the meaning of a derivative value in an applied problem

## Key Vocabulary
- Derivative, difference quotient, limit definition of the derivative
- Differentiability, non-differentiable, corner, cusp, vertical tangent
- Power rule, product rule, quotient rule
- Tangent line, instantaneous rate of change
- Slope of secant line, average rate of change
