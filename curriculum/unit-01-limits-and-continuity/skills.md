# Unit 1: Limits and Continuity — Skills

## AP Calculus AB Skill Categories

### Skill 1: Implementing Mathematical Processes
- Estimate limit values numerically using a table of values
- Evaluate limits algebraically (factor, cancel, rationalize)
- Determine one-sided limits from piecewise-defined functions
- Apply the definition of continuity to classify a function at a point

### Skill 2: Connecting Representations
- Translate between graphical and algebraic representations of limits
- Identify asymptotic behavior from a graph and write the corresponding limit statement
- Interpret a table of values to describe limit behavior

### Skill 3: Justification
- Use limit notation to justify whether a limit exists
- Justify continuity or discontinuity using the three-part definition
- Apply the Intermediate Value Theorem with a written justification

### Skill 4: Communication and Notation
- Use correct limit notation throughout solutions
- Write clear mathematical justifications for continuity claims
- Express asymptotic behavior precisely in limit form

## Key Vocabulary
- Limit, limit notation, one-sided limit
- Continuity, removable discontinuity, jump discontinuity, essential (infinite) discontinuity
- Vertical asymptote, horizontal asymptote
- Intermediate Value Theorem (IVT)
- Squeeze Theorem
