# Unit 1: Limits and Continuity — Practice

## Starter Practice Problems

### Estimating Limits from Tables
1. Use the table below to estimate lim(x→2) f(x).

| x    | 1.9  | 1.99 | 1.999 | 2.001 | 2.01 | 2.1  |
|------|------|------|-------|-------|------|------|
| f(x) | 3.61 | 3.96 | 3.996 | 4.004 | 4.04 | 4.41 |

### Limits from Graphs
2. Given the graph of f(x), determine:
   - lim(x→3⁻) f(x)
   - lim(x→3⁺) f(x)
   - lim(x→3) f(x) (if it exists)
   - f(3)

### Algebraic Limits
3. Evaluate: lim(x→5) (x² - 25) / (x - 5)
4. Evaluate: lim(x→0) (√(x+4) - 2) / x
5. Evaluate: lim(x→∞) (3x² - 2x + 1) / (x² + 5)

### One-Sided Limits
6. For f(x) = { x + 1, x < 2; x² - 1, x ≥ 2 }, find:
   - lim(x→2⁻) f(x)
   - lim(x→2⁺) f(x)
   - Is f continuous at x = 2? Justify.

### Continuity
7. Determine whether f(x) = (x² - 9) / (x - 3) is continuous at x = 3. Classify any discontinuity.
8. Find the value of k that makes f continuous: f(x) = { kx + 2, x < 4; x² - 6, x ≥ 4 }

### Intermediate Value Theorem
9. Show that f(x) = x³ - x - 1 has a zero on the interval [1, 2] using the IVT.
10. Explain why the IVT does or does not apply to f(x) = 1/x on [-1, 1].

## Activity Ideas
- Graph investigation: sketch a function that satisfies given one-sided limit conditions
- Limit card sort: match limit notation, graphs, and tables
- Continuity station rotation: classify functions as continuous or discontinuous
- IVT real-world context: use temperature data to argue that a specific value was reached
