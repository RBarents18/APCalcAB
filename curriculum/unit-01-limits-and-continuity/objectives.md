# Unit 1: Limits and Continuity — Learning Objectives

By the end of this unit, students will be able to:

1. Interpret and estimate limit values from tables and graphs.
2. Use limit notation correctly to describe function behavior.
3. Evaluate limits algebraically using factoring, rationalization, and other techniques.
4. Identify and evaluate one-sided limits.
5. Determine whether a limit exists and explain why or why not.
6. Describe behavior of functions as x approaches infinity or negative infinity.
7. Identify vertical and horizontal asymptotes using limits.
8. Define continuity formally using limit language.
9. Determine whether a function is continuous at a point using the three-part definition.
10. Classify discontinuities as removable, jump, or essential.
11. Determine intervals of continuity from graphs and equations.
12. Apply the Intermediate Value Theorem to verify the existence of a value.
